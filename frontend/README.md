# Knowra Web Interface

A premium, glassmorphism-inspired chat interface for interacting with the Knowra RAG system.

## Experience
- **Fluid Animations**: Powered by Framer Motion.
- **Glassmorphism Design**: High-transparency cards with backdrop blurs.
- **Verified Sources**: Unique "Source Badges" with hover-enabled data previews.
- **Dark/Light Mode**: Full theme support with synchronized storage.
- **Auto-expanding Input**: Intelligent textarea for long queries.

## Development
- **Vite/React**: Modern build tool and framework.
- **Tailwind CSS 4.0**: Utility-first styling with the latest features.

## Environment Variables
- `VITE_API_BASE_URL`: The URL of your Knowra Backend (Render or Localhost).

## Deployment (Vercel)
The project is optimized for Vercel with a `vercel.json` for SPA routing support.
